8.1 全体像：3種の行政手続きと発生タイミング

クリニック立ち上げの9工程のうち、行政系手続きは最も機械的だが、順番を間違えると物件契約のやり直し・開業遅延・ベンダー審査落ちに直結する。本章は日本国内の新規クリニック開設を前提に、手続きのクリティカルパスと、TEN EXPERIENCE社内で実際に発生した落とし穴をまとめる。

新規クリニック開設で絡む行政手続きは大きく3系統:

物件候補決定 → 賃貸契約 → 内装工事 → 開業届 → 開業 │ │ │ │ ① 保健所 ② 法人 ③ 構造設備 ④ 保険医療機関 事前相談 設立確定 検査立会 指定（保険診療時） + 医療廃棄物 処理契約
#手続き所轄タイミングポイント
①保健所 事前相談管轄保健所（例: 台東区）賃貸契約前契約後に構造設備NGが出ると物件やり直し
②法人設立（一般社団法人 or 医療法人）法務局物件決定と並走一社を選ぶ場合は「なぜ医療法人でないか」の理由書必須
③医療廃棄物処理契約産業廃棄物処理業者開業日前契約名義（屋号 or 法人名）を事前確定
④厚生局 保険医療機関指定地方厚生局開業後（保険診療時のみ）自由診療のみなら届出先なし。外部ベンダーに説明テンプレが必要
重要: ①→②→③→④ は順次ではなく並走。特に①は物件決定直後に即動く。

8.2 物件候補が見つかったら：保健所に即アポを取る（R-G-003）

手順

  1. 物件候補が定まったら、賃貸契約を結ぶ前に管轄保健所に電話
  2. 以下3段階の手続きを確認:
    • 開設許可（一般社団法人の場合は必須。結構時間がかかる）
    • 開設届
    • 保険医療機関指定（保険診療を行う場合）
  3. 保健所から「賃貸契約前に相談に来てほしい」と言われたら即アポ取り
  4. 東先生（医師）と嶋田さん（事業開発）同席で訪問
  5. 物件の構造設備を事前チェック:
    • 部屋サイズ
    • 導線（患者動線・医療従事者動線）
    • 水場
    • 医薬品保管スペース

落とし穴

契約後に保健所から構造設備でNGが出ると、物件選びからやり直しになる。特に23区内（台東区等）では賃貸契約前相談を明示的に要請される。

簡略化できるケース

居抜き物件で、前クリニックと同一要件（同診療科・同規模）の場合は簡略化可。ただし保健所に電話での確認は必ず行う。

8.3 法人形態の判断：一般社団法人を選ぶ場合の準備（R-G-001）

判断フロー

Q. 運営法人の形態はどうするか？ ├─ 医療法人 │ └─ 標準的。保険医療は問題なく動く │ └─ 一般社団法人 └─ 選ぶ場合: 保健所への「理由書」必須 ├─ 「なぜ医療法人ではなく一般社団法人か」を明文化 ├─ 事業計画書・定款と整合を取る └─ 保健所の開設許可審査で参照される

一般社団法人を選ぶ場合のチェックリスト

  • 「なぜ医療法人ではないか」の理由書ドラフト作成
  • 事業計画書の運営主体記載を一般社団法人名義で統一
  • 定款の事業目的が医療提供と整合しているか確認
  • 保健所開設許可の審査期間を織り込んだスケジュール（通常より長い）

落とし穴

「理由書」を後回しにすると保健所審査で差し戻しが発生し、開業日が後ろ倒しになる。物件決定と並走で準備開始すべき。

8.4 開業日前までに：医療廃棄物処理業者と契約（R-G-005）

手順

  1. 複数業者から見積取得（容器代・回収運搬費・処理費）
  2. 契約名義を明示的に確認:
    • クリニック屋号単体（例: 「さくらインターナショナルクリニック東京」）
    • 一般社団法人名義（例: 「一般社団法人 SIG」）
  3. 開業日前に契約締結、初回回収日を確定

名義判断の観点

名義メリットデメリット
クリニック屋号単体請求書の計上が店舗単位で明瞭複数院展開時に契約を店舗ごとに巻き直す必要
一般社団法人複数院を一括契約にまとめられる店舗別の経理計上でひと手間

落とし穴

契約名義を曖昧にしたまま開業すると、請求書発行・経理計上・医療法上の管理責任の紐付けで齟齬が出る。開業後に廃棄物が実際に発生する前に確定必須。

例外

訪問診療のみで医療廃棄物が発生しない運用なら、開業初月は後回しも可。

8.5 運用開始後：外部ベンダー審査で「厚生局許可証」を求められたとき（R-G-004）

決済会社・配送会社・ベンダーの審査書類で、しばしば「厚生局許可証」等の提出を求められる。保険診療を前提にした要求なので、自由診療クリニックは正面から説明する必要がある。

回答テンプレ

【自由診療のみの場合】

当院は自由診療のみを提供しており、保険診療は行っておりません。そのため厚生局への届出先は特になく、許可証等も発行されません。

根拠: 健康保険法上、保険医療機関指定申請は保険診療を行う場合に必要な手続きであり、自由診療のみの医療機関には適用されません。

（必要に応じて管轄保健所への電話確認内容を議事メモ形式で添付）

実運用（過去事例）

  • 2025年8月 SBペイメント審査: 上記テンプレで通過
  • 相手の審査担当は保険診療を前提にしていることが多いため、簡潔な議事メモを添えると説明コストが下がる

落とし穴

説明なしで「許可証がない」状態で放置すると、審査落ちまたは大幅な遅延になる。新規ベンダー審査は必ずこのテンプレから入る。

保険診療併設時

保険診療を併設する場合は厚生局届出が実際に必要なので、届出済み証跡を素直に提出する。

8.6 落とし穴サマリー（章末チェックリスト）

物件決定 → 賃貸契約の前に

  • 管轄保健所に電話して3段階手続きを確認したか
  • 事前相談アポを取ったか（東先生・嶋田さん同席予定か）
  • 構造設備の要件を物件で満たせるか事前確認したか
  • 一般社団法人を選ぶ場合、理由書ドラフトを準備開始したか

開業日の2週間前までに

  • 医療廃棄物処理業者と契約名義を確定したか
  • 初回回収日を確定したか
  • 保健所開設許可・開設届の提出が完了したか
  • 保険診療を行う場合、厚生局への保険医療機関指定申請を進めているか

運用開始後

  • 外部ベンダー審査用の「厚生局届出要否」説明テンプレを手元に用意しているか
  • 新規ベンダー審査で毎回説明が発生するため、テンプレを最新化しているか

一般化の注意

  • 自治体差あり: 8.2（保健所事前相談）は台東区での実運用確認。渋谷区・港区等では手続きの詳細が異なる可能性
  • 保険 vs 自費: 8.5（厚生局届出要否）は自由診療のみを想定。保険診療併設時は届出が必要
  • 立地固有: 8.2の「賃貸契約前相談要請」は23区内で頻出。地方部の運用は未検証

本サンプルが示すこと

この章を読むと、クリニック立ち上げ担当者が実際に手を動かす順序がわかる:

  1. 物件候補を見つけたら保健所に電話（8.2）
  2. 法人形態を決定し、一般社団なら理由書着手（8.3）
  3. 開業日前に医療廃棄物業者と契約（8.4）
  4. 運用開始後のベンダー審査対応テンプレを手元に（8.5）

26ルールの中でもR-G系列（行政・申請）は★3が集中し、個別案件固有度が低いため、このレベルの章がいますぐ書ける。

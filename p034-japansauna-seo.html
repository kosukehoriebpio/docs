自宅サウナの間取りに必要な基本知識

自宅サウナを検討する際、まず理解しておくべきなのが「サウナ体験を構成する3つの要素」と「それぞれに必要なスペース」です。ここでは間取りを考える前提として押さえておきたい基礎知識を解説します。

サウナ体験を構成する3つのゾーン

  1. サウナ室（ホットゾーン）: 高温で発汗を促すメインの空間です。自宅用では乾式（ドライサウナ）とスチーム式（ミストサウナ）の2種類が一般的です。
  2. 水風呂・冷水シャワー（コールドゾーン）: サウナ後に身体を冷やすための空間です。専用の水風呂を設置する方法と、既存の浴室シャワーを活用する方法があります。
  3. ととのいスペース（レストゾーン）: 加温と冷却の後に身体を休める空間です。外気浴ができるバルコニーやテラス、または室内の休憩スペースが該当します。

この3つのゾーンをいかにスムーズな動線で結ぶかが、自宅サウナの間取り設計における最大のポイントとなります。

サウナ室の広さの目安

  • 1人用: 約0.75畳（幅0.9m x 奥行1.2m程度）。最小限のスペースで、座って利用する形式です。
  • 2人用: 約1〜1.5畳（幅1.2m x 奥行1.5m程度）。夫婦やパートナーと一緒に利用できるサイズです。
  • 3〜4人用: 約2〜3畳（幅1.8m x 奥行2.0m程度）。家族やゲストとの利用も想定した余裕のあるサイズです。

なお、建築基準法では居室の天井高は2.1m以上と定められていますが、サウナ室は一般的に「居室」には該当しないと解釈されるケースが多く、天井高を1.8〜2.0m程度に抑えることで効率的に室温を上げることが可能です（※自治体により解釈が異なる場合があるため、設計段階で必ず確認してください）。また、換気設備については建築基準法第28条に基づく換気基準への適合が求められる場合がありますので、設計時に専門家への相談をおすすめします。

[画像: サウナ室の広さ別（1人用・2人用・4人用）の平面寸法図。各サイズの畳数とメートル寸法を図示]

水風呂・ととのいスペースに必要な面積

  • 水風呂: 専用の水風呂を設置する場合、最低でも幅0.7m x 奥行1.0m x 深さ0.5m程度のスペースが必要です。既存の浴槽を冷水で満たして代用する方法もあります。水風呂の設置について詳しくは、自宅に水風呂を設置するポイントをご覧ください。
  • ととのいスペース: リクライニングチェアやインフィニティチェア（背もたれが大きく倒れるアウトドアチェア）を1脚置く場合、約1畳（1.8m x 0.9m）程度は確保したいところです。外気浴を楽しむなら、バルコニーやテラスに2畳以上のスペースがあると快適です。

電気容量と設備面の前提条件

  • 家庭用電気サウナ: 一般的に100V/200Vで消費電力は2〜6kW程度。既存の電気契約容量（一般家庭で40〜60A）では不足する場合があり、契約変更や分電盤の増設が必要になることがあります。
  • 排水設備: 水風呂を設置する場合は、排水経路の確保が必要です。既存の浴室に隣接させることで排水工事のコストを抑えられます。
  • 断熱・防水: サウナ室周辺は高温多湿になるため、適切な断熱材と防水処理が不可欠です。

失敗しない自宅サウナの間取り設計 5つのステップ

ステップ1 ─ サウナの種類と設置場所を決める

まず最初に決めるのは、サウナの種類と設置場所です。

サウナ種類特徴必要スペース間取りへの影響
屋内据え置き型既存の室内に設置1〜3畳+周囲の余裕浴室隣接が理想
屋内造作型建築時に組み込む1〜4畳設計自由度が高い
屋外独立型庭・テラスに設置設置面積+動線スペース母屋との動線が課題
バレルサウナ庭に樽型を設置直径2m前後の円形搬入経路の確保が必要

設置場所の候補:

  • 浴室隣接型: 水回り設備を共有できるため、コストを抑えやすい。リフォームで特に有効。詳しくはリフォームでサウナ導入する方法をご参照ください。
  • ガレージ・地下室型: 防音性が高く、生活空間への影響が少ない。
  • 庭・テラス型: 外気浴と組み合わせやすい。詳細は屋外サウナの設置方法で解説。

ステップ2 ─ 3ゾーンの配置を決める

配置の基本原則:

  1. 直線動線の確保: サウナ室 → 水風呂 → ととのいスペースの順に、できるだけ直線的に移動できる配置が理想。
  2. 水風呂はサウナ室の隣に: サウナ室と水風呂（冷水シャワー）は隣接させるのが鉄則。
  3. ととのいスペースは最後に: 水風呂で冷やした後、すぐに休憩できる場所を確保。

推奨する配置パターン:

  • パターンA（浴室一体型）: 浴室の隣にサウナ室を設け、浴室の浴槽を水風呂として活用。ととのいスペースは脱衣所または隣接するバルコニーに設置。
  • パターンB（屋外連続型）: 庭にサウナ小屋を設置し、隣に水風呂（チラー付き）を配置。ウッドデッキ上にととのいチェアを設置。
  • パターンC（地下室活用型）: 地下室にサウナ室と水風呂を設け、1階テラスに出てととのいスペースとする。

ステップ3 ─ サウナ動線を最適化する

  • 移動距離の最短化: サウナ室から水風呂まで3〜5歩（2〜3m以内）が理想。
  • 段差の排除: 転倒防止の観点からフラットな動線を確保。
  • 滑りにくい床材: タイルやテラコッタなど水に強く滑りにくい素材を選択。
  • プライバシーの確保: 家族の生活動線や外部からの視線と交差しない設計が重要。

ステップ4 ─ 換気・排水・電気の設備動線を計画する

換気計画: サウナ室の換気回数は1時間あたり3〜6回が目安。吸気口はヒーター付近の低い位置に、排気口は対角の高い位置に設けるのが一般的。

排水計画: 水風呂を設置する場合、排水管の勾配確保が重要。既存の浴室排水に接続する場合は、配管距離が短くなるよう間取りを工夫。

電気設備計画: 200V電源が必要な場合は、分電盤からサウナ室まで専用回路を引く必要があります。

ステップ5 ─ 家族の生活動線との両立を確認する

  • サウナ動線と生活動線の分離: リビングやキッチンを通らずに移動できるルートを確保。
  • 音の問題: サウナヒーターの稼働音やロウリュ時の音が寝室やリビングに響かないよう防音対策。
  • 湿気の影響: サウナ室周辺の湿気が他の居室に影響しないよう換気経路と防湿処理を計画。
  • 来客時の動線: ゲストをサウナに招く可能性がある場合は、玄関からサウナまでの専用動線を検討。

自宅サウナの間取りで失敗しないための注意点

失敗例1 ─ ととのいスペースを後回しにした

最も多い失敗が、ととのいスペースの確保を後回しにしてしまうケースです。

対策: 間取り設計の初期段階から、ととのいスペースを3ゾーンの1つとして明確に位置づけ、最低1畳以上のスペースを確保しましょう。

失敗例2 ─ 水回りから離れた場所に設置した

対策: 可能な限り既存の浴室・洗面所に隣接した場所を選びましょう。水回りを集約した間取りにすることで配管工事のコストを大幅に抑えられます。

失敗例3 ─ 将来の変更を想定していなかった

対策: 壁を取り外せる構造にしておく、配管のための下地を先行して施工しておく、電気容量に余裕を持たせるなどの工夫が有効です。

失敗例4 ─ 近隣への配慮が不足していた

対策: 屋外設置の場合は、設置位置を隣地境界から十分に離し、煙突の向きや排気方向を考慮した間取りにしましょう。
